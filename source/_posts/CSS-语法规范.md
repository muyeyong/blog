# css语法规范

## 样式格式书写

> 用宽松式风格

```css
error: p {color: red;fond-size:20px;}

right: p {
  color: red;t
  font-size: 20px;
}
```



## 样式大小写书写风格

> 使用小写

```css
error: p {
    COLOR: RED;
}

right: p {
    color: red;
}
```



## 样式空格风格

> 选择器和大括号中间保留空格
>
> 属性值前面，冒号后面保留空格

```
p {
	color: red;
}
```
